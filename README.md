# 🚢 1TERMINAL JICT, MALT, KOJA, NPCT - Portfolio

## 📊 Project Overview

- **Real-time Shipment Tracking**
- **Performance Analytics**
- **Automated Reporting**
- **Interactive Dashboards**
- **Data Visualization**
